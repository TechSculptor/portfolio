### Description
A simple web application for medical appointment booking, developed to demonstrate skills in PHP, database management (PostgreSQL), and containerization (Docker).

### Features
- **Authentication**: Login/Register system for patients.
- **Roles**: Distinct roles for Patients, Doctors, and Administrators.
- **Appointment Booking**: Patients can book time slots.
- **Doctor Management**: Administrators can add, edit, or delete doctors.
- **Availability**: Dynamic display of available slots.
- **Internationalization**: Multi-language support (French/English).

### Quick Start

1.  **Prerequisites**: Docker and Docker Compose installed.
2.  **Launch**:
    ```bash
    docker-compose up --build
    ```
